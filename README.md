# archery-score-recorder

A web app for recording and tracking archery scores across competitions.

## Tech Stack

- **Frontend/Backend** — Next.js 15 (App Router, Server Actions)
- **Database** — PostgreSQL via [Neon](https://neon.tech)
- **UI** — shadcn/ui + Tailwind CSS

## Features

- Create and record scores for archery competitions
- End-by-end arrow entry with enforced descending order (X → M)
- View score results and recent history
